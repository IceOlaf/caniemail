margin logical properties - inline & block

<div style="margin-inline: 20px 10px;">
Lorem ipsum.
Lorem ipsum.
<div style="margin-inline: 20px 10px;" dir="rtl">
Lorem ipsum.
Lorem ipsum.
<div style="margin-block: 20px;">
Lorem ipsum.
<div style="margin-inline-start: 30px;">
Lorem ipsum.
<div style="margin-inline-end: 30px;">
Lorem ipsum.
<div style="margin-block-start: 30px;">
Lorem ipsum.
<div style="margin-block-end: 30px;">
Lorem ipsum.

The green color denotes the space left by the margin. If we see only orange, then margin logical properties are not supported.
